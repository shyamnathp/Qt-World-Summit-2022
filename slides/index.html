QtScrypt (1/2)

  • Inspired by QtScript, not a port
  • A dynamic way to interact with Python from C++
  • Enabling Python modules within C++ 🐍
  • proof of concept


// Simple Engine
QScryptEngine e;
e.execute(...);
QVariant r = e.evaluate(...);

// Load Python module
QScryptModule mod(...);
// Get Function Reference
QScryptFunction f(..., &mod;);

// Call Function
QVariantList args;
QVariant r = f.call(args);
